# Database Design

## Database Strategy

### Database Architecture

The platform follows a polyglot persistence strategy.

### Primary Database

PostgreSQL

Used for:

* Organizations
* Users
* Projects
* Test Cases
* Test Plans
* Executions
* Reports
* Audit Logs

### Cache Layer

Redis

Used for:

* Sessions
* Authentication Tokens
* Job Queues
* Frequently Accessed Data

### Search Layer

Elasticsearch

Used for:

* Full Text Search
* Log Search
* Execution Search
* Reporting Analytics

### Object Storage

S3 Compatible Storage

Used for:

* Screenshots
* Videos
* Attachments
* Exported Reports

### Vector Database

Qdrant

Used for:

* AI Knowledge Base
* Test Generation Context
* Historical Test Analysis
* Semantic Search

# Multi Tenant Design

## Tenant Model

Shared Database

Tenant Isolation via organization_id

Every business table contains:

* id
* organization_id
* created_at
* updated_at

Example:

organizations
projects
test_cases
executions
reports

All queries automatically filtered by organization_id.

---

## Tenant Security Rules

User can only access records belonging to their organization.

Backend middleware validates:

* JWT
* organization_id
* permissions

Cross tenant access prohibited.

# Tables

Represents a customer workspace.

* id UUID PK
* name VARCHAR(255)

* description TEXT

# Indexing Strategy

## Query Optimization

Create indexes for:

* organization_id
* project_id
* status
* created_at
* updated_at

Composite Indexes:

(project_id, status)

(project_id, created_at)

(execution_id, status)

(organization_id, created_at)

---

## Partitioning Strategy

Large tables partitioned by month.

Tables:

* executions
* execution_steps
* audit_logs

Example:

executions_2026_01

executions_2026_02

executions_2026_03

Benefits:

* Faster queries
* Easier maintenance
* Reduced index size

---

## Archiving Strategy

Retention Rules

Execution Logs:

* 12 months hot storage
* 24 months archive storage

Audit Logs:

* 7 years retention

Reports:

* 24 months retention

Screenshots/Videos:

* Lifecycle policy enabled

Archive Storage:

* S3 Glacier
* Cold Object Storage

Benefits:

* Lower storage cost
* Regulatory compliance
* Improved database performance
